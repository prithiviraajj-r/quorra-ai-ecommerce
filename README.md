📌 About Project

Quorra AI Marketplace is a modern e-commerce web application that helps users search products, filter items, manage their cart, and get AI-based product recommendations — all in a clean, responsive interface.


✨ Features

FeatureDescription🤖 AI Shopping AssistantSmart product recommendations powered by AI🔍 Product SearchFind items instantly by name or keyword🛍️ Product FilteringNarrow results by category, price, and more⭐ Product RatingsStar ratings to guide purchase decisions🛒 Add to CartSeamless cart management and item control💰 Price CalculationReal-time totals and order summaries📱 Responsive DesignWorks beautifully on all screen sizes
